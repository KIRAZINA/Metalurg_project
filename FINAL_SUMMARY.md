## 🎯 Что было реализовано?

Полнофункциональный модуль оптимизации для определения **минимальных входных параметров примесей** на основе регрессионных моделей, решающий обратную задачу:

**Задача:**  
Найти минимальное количество серы (S) и кремния (Si) в исходных материалах, которые обеспечат максимально чистую готовую сталь.

**Решение:**  
Использование линейной регрессии в **обратном направлении** + многокритериальная оптимизация (Парето-фронт).

## 🎓 Математические основы

### Обратная регрессия

**Прямая модель (OLS):**
```
Y = a + b × X
```

**Обратная функция:**
```
X = (Y - a) / b
```

### Парето-оптимальность

Решение A доминирует B, если:
- `sum(A.inputs) ≤ sum(B.inputs)` **AND**
- `sum(A.outputs) ≤ sum(B.outputs)` **AND**
- хотя бы одно неравенство строгое

**Парето-фронт** = все недоминируемые решения = оптимальные компромиссы
